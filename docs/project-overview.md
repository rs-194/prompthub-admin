# 项目概览（Project Overview）

## 1. 项目背景
PromptHub Admin 是一个基于 Vue3 + TypeScript + Vite + Element Plus 的 AI 提示词与知识库管理后台项目。

该项目当前不是完整商业产品，而是用于学习、作品集和 AI 应用后台能力展示的个人项目。

## 2. 项目目标
- 搭建清晰、可扩展的后台前端工程结构。
- 支撑后续提示词管理、模型配置、知识库管理与对话测试等 AI 应用场景。
- 逐步接入 FastAPI 后端接口，形成可演示的端到端管理后台。

## 3. 核心模块
- Dashboard 首页（数据看板）
- 提示词管理
- 知识库管理
- 模型配置
- 对话测试
- 系统设置

说明：当前多数模块仍处于页面骨架阶段，业务逻辑待实现。

## 4. 面向的使用场景
- 个人 AI 项目后台管理界面实践。
- 面试与作品集展示中的工程组织与前端架构能力说明。
- 后续作为 AI 管理后台原型进行功能扩展与联调。

## 5. MVP 范围
当前 MVP 重点：
- 前端后台系统骨架
- 基础布局与导航
- 页面路由与菜单结构
- 后续 AI 应用管理能力扩展的前置准备

## 6. 非目标范围
当前阶段不包含：
- 完整商业级权限体系
- 完整业务闭环与生产级运营能力
- 全量后端服务与真实数据闭环

说明：后端待接入，前端业务功能将分阶段补齐。
